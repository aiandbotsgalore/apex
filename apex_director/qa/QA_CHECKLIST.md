# APEX DIRECTOR - COMPREHENSIVE QA CHECKLIST

## Executive Summary
This checklist ensures professional quality standards for all video content produced through the APEX DIRECTOR platform. All items must be validated before content delivery.

---

## 1. PRE-PRODUCTION VALIDATION

### Technical Specifications
- [ ] **Source Material Quality Check**
  - [ ] Resolution meets project requirements (4K, 1080p, etc.)
  - [ ] Frame rate consistent throughout (24fps, 30fps, 60fps)
  - [ ] Color space properly configured (Rec.709, Rec.2020)
  - [ ] No corrupted or missing frames detected
  - [ ] Audio sample rate and bit depth verified

- [ ] **File Format Compliance**
  - [ ] Container format meets delivery specifications
  - [ ] Codec settings optimized for target platform
  - [ ] File size within acceptable limits
  - [ ] Metadata correctly populated

- [ ] **Rights and Clearances**
  - [ ] All music tracks properly licensed
  - [ ] Talent release forms obtained and filed
  - [ ] Location permits secured
  - [ ] Trademark clearances completed
  - [ ] Copyright notices verified

---

## 2. VISUAL QUALITY ASSESSMENT

### Color and Exposure
- [ ] **IRE Level Compliance** (7.5 - 100 IRE)
  - [ ] No clipping in highlights (above 100 IRE)
  - [ ] No crushing in shadows (below 7.5 IRE setup)
  - [ ] Consistent exposure throughout sequence
  - [ ] Legal broadcast levels maintained

- [ ] **Color Accuracy**
  - [ ] White balance consistent across shots
  - [ ] Color temperature appropriate for scene
  - [ ] Skin tones appear natural and accurate
  - [ ] Color grading consistent throughout
  - [ ] No color casts or unwanted tints

### Visual Consistency
- [ ] **Style Continuity**
  - [ ] Consistent visual style across all shots
  - [ ] No style drift between sequences
  - [ ] Color grading matches reference materials
  - [ ] Lighting style consistent with director's vision
  - [ ] Camera movement and framing consistent

- [ ] **Motion Quality**
  - [ ] No motion blur (except where intentional)
  - [ ] Smooth camera movements
  - [ ] No stuttering or frame drops
  - [ ] Consistent frame rate throughout
  - [ ] Stable footage where required

### Composition and Framing
- [ ] **Rule of Thirds Compliance**
  - [ ] Key elements positioned on thirds lines
  - [ ] Subject matter properly framed
  - [ ] Headroom appropriate for content type
  - [ ] Edge composition aesthetically pleasing

- [ ] **Safe Area Compliance**
  - [ ] Important content within action safe area (90% of frame)
  - [ ] Text and graphics within title safe area (80% of frame)
  - [ ] No critical elements cut off at edges
  - [ ] Proper margins for different aspect ratios

---

## 3. AUDIO QUALITY VALIDATION

### Audio Levels
- [ ] **Broadcast Standards** (-23 LUFS for broadcast)
  - [ ] Dialogue levels properly balanced
  - [ ] Music and effects appropriately mixed
  - [ ] No clipping or distortion in any track
  - [ ] Dynamic range suitable for delivery format
  - [ ] Consistent levels throughout content

### Audio Clarity
- [ ] **Dialogue Quality**
  - [ ] Clear, intelligible speech throughout
  - [ ] No background noise or interference
  - [ ] Consistent voice quality across shots
  - [ ] Proper microphone technique maintained
  - [ ] ADR (Automated Dialogue Replacement) if needed

- [ ] **Sound Design**
  - [ ] Effects properly timed and balanced
  - [ ] Ambient sound supports scene mood
  - [ ] No audio artifacts or digital glitches
  - [ ] Appropriate reverb and processing
  - [ ] Creative audio elements enhance storytelling

---

## 4. SYNCHRONIZATION VERIFICATION

### Audio-Video Sync
- [ ] **Frame-Accurate Timing**
  - [ ] Audio matches video within ±40ms tolerance
  - [ ] Lip sync accurate for all dialogue
  - [ ] Sound effects properly synchronized
  - [ ] Music cues timed correctly
  - [ ] No sync drift over content duration

- [ ] **Technical Sync Validation**
  - [ ] Frame rate consistency maintained
  - [ ] No dropped or duplicated frames
  - [ ] Timecode continuous and accurate
  - [ ] Sync markers properly placed
  - [ ] Multicam sync where applicable

## 5. BROADCAST STANDARDS COMPLIANCE

### Legal Broadcast Levels
- [ ] **IRE Compliance**
  - [ ] Setup level: 7.5 IRE minimum
  - [ ] Peak level: 100 IRE maximum
  - [ ] No illegal signal levels detected
  - [ ] Luminance within broadcast standards
  - [ ] Chrominance levels properly limited

- [ ] **Color Gamut Validation**
  - [ ] No out-of-gamut colors detected
  - [ ] Broadcast-safe color space maintained
  - [ ] Proper saturation levels throughout
  - [ ] No color clipping or bleeding
  - [ ] Consistent color representation

### Professional Standards
- [ ] **Encoding Quality**
  - [ ] Appropriate compression settings
  - [ ] No visible compression artifacts
  - [ ] Proper bitrate allocation
  - [ ] Optimal codec settings for delivery
  - [ ] File integrity verified

- [ ] **Delivery Specifications**

  - [ ] Aspect ratio correct for platform
  - [ ] Closed captioning if required
  - [ ] Audio description available if needed
  - [ ] Metadata properly formatted
